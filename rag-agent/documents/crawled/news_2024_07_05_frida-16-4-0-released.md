##  Frida 16.4.0 Released [∞](https://frida.re/news/2024/07/05/frida-16-4-0-released/ "Permalink")
release
05 Jul 2024  [ ![](https://github.com/oleavr.png) oleavr ](https://github.com/oleavr)
This release is packing some exciting new things. Let’s dive right in.
## CoreDevice
As mentioned in the 16.3.0 release notes, [@hsorbo](https://twitter.com/hsorbo) and I were working on submitting a patch for the Linux kernel’s CDC-NCM driver to make it compatible with Apple’s private network interface. This has since gone [upstream](https://github.com/torvalds/linux/commit/3ec8d7572a69d142d49f52b28ce8d84e5fef9131), and will be part of Linux 6.11.
In the meantime, and for those of you using Frida on Windows, we have just implemented a [minimal user-mode driver](https://github.com/frida/frida-core/blob/31188db39a7c9ae24f640a34b3fdf701f4a93bb3/src/fruity/ncm.vala) that Frida now uses when it detects that the kernel doesn’t provide one. We leveraged [lwIP](https://savannah.nongnu.org/projects/lwip/) to also do Ethernet and IPv6 entirely in user space. The result is that Frida can support CoreDevice on any platform supported by libusb.
