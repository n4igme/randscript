Randscript / rag-agent

RAG Agent

Local Retrieval Augmented Generation system with Ollama. Ingest documents, crawl websites, and chat with your knowledge base — entirely offline.

Local LLM Semantic Search Multi-Format Offline
💬

Chat Interface

Web UI and CLI with real-time streaming responses. Ask questions and get answers with source citations from your knowledge base.

📄

Document Ingestion

Ingest PDF, Word, PowerPoint, Excel, HTML, Markdown, and text files. Automatic format detection and conversion via Docling.

🎤

Audio Transcription

Transcribe MP3, WAV, and M4A files using Whisper ASR. Audio content becomes fully searchable through the RAG pipeline.

🕸️

Web Crawler

Crawl documentation sites and ingest them into your knowledge base. Supports sitemaps, recursive crawling, and llms.txt.

🧠

Semantic Chunking

LLM-driven semantic splitting with rule-based fallback. Configurable chunk sizes for optimal retrieval performance.

🔒

Fully Local

Runs entirely on your machine with Ollama. No API keys required, no data leaves your network. Complete privacy.

Quick Start

# 1. Install dependencies cd rag-agent uv sync && cp .env.example .env # 2. Configure .env with DATABASE_URL and Ollama settings # 3. Set up database psql $DATABASE_URL < sql/schema.sql # 4. Pull Ollama models ollama pull mistral ollama pull nomic-embed-text # 5. Start the web interface uv run python web_app.py # http://localhost:8000 # Or use the CLI uv run python cli.py # 6. Ingest documents uv run python -m ingestion.ingest --documents documents/

Docker Deployment

# Start all services docker-compose up -d # Ingest documents docker-compose --profile ingestion up ingestion

LLM Provider Options

Ollama (Local)

Run entirely offline. No API costs.

LLM: mistral, llama3.2, qwen2.5, deepseek-r1

Embeddings: nomic-embed-text, mxbai-embed-large

OpenAI (Cloud)

Higher quality, requires API key.

LLM: gpt-4o-mini, gpt-4o

Embeddings: text-embedding-3-small
